File store settings in json format.

Name: Example..
Menu URL: http://example.com/dir
Multi-Page: next page css
Page CSS: .className .... li a(all the way to the link path)
Page Link: http://example.com/123456/555.html then the link will be http://example.com
Title CSS: .className .... h1 (Title of the page)
Body CSS: .className .... div.content (Content of the page)
UnwantedSelector: .className .... div.footer (Unwanted selector)
